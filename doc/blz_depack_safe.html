<brieflz.h> - blz_depack_safe
size_t blz_depack_safe( const void *source,
                        size_t srclen,
                        void *destination,
                        size_t depacked_size );

Decompresses the compressed data from source[] into destination[].

The destination[] buffer must be large enough to hold the decompressed data.

The depacked_size parameter is the size of the original uncompressed data.

This function reads at most srclen bytes from source[], and writes at most depacked_size bytes to destination[]. If there is not enough source or destination space, or a decoding error occurs, the function returns BLZ_ERROR.

Parameters:
source - pointer to the compressed data.
srclen - the size of the source buffer in bytes.
destination - pointer to where the decompressed data should be stored.
depacked_size - the size in bytes of the decompressed data.
Returns:
the length of the decompressed data, or BLZ_ERROR on error.
